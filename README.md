# KSR-ATT85
Radio with an ATTINY85

## List of electronic components
- MCU - ***ATTINY85***
- Screen - ***IIC OLED SSD1306***
- Regulator - ***LM1117-3.3***
- FM receiver - ***TEA5767*** (module)
- potentiometer - ***b103*** (or similar)
- 5V power supply
- Speaker

## Libraries
- SPI
- [KS_TEA5767](https://github.com/kerogs/KS_TEA5767)
- Wire
- Adafruit SSD1306
- Adafruit GFX
